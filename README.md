# HTML and CSS Course Projects

This repository contains all the web development projects and exercises completed during my HTML and CSS course. Each project is organized in its own folder and demonstrates a progression from foundational HTML structure to advanced responsive layouts.

---

## 🚀 Projects

### 1. Lumina Agency Landing Page — Bootstrap 4
**Folder:** `bootstrap4_project/` (root `index.html`)

A premium creative agency website built with **Bootstrap 4**, HTML5, CSS3, and vanilla JavaScript.

**Key Features:**
- Responsive **Navbar** with collapsible mobile menu
- Professional grid layout using Bootstrap's `container`, `row`, and `col` classes
- **Bootstrap Cards** for service offerings, portfolio displays, and team members
- Stylized **contact form** with Bootstrap validation classes
- Custom aesthetics: glassmorphism effects, smooth animations, and Google Fonts (Outfit & Inter)

---

### 2. The Space Station — Gallery Page
**Folder:** `Project/`

A themed resort website for a fictional space station, featuring an interactive image gallery.

**Key Features:**
- Sticky navigation bar with Home, Gallery, and Reservations links
- Responsive **CSS Grid** photo gallery with 9 space-themed thumbnail images
- Clean, card-style image layout with rounded corners
- Semantic HTML5 structure with a `<nav>`, `<footer>`, and structured sections

---

### 3. One-Page Website
**Folder:** `One-Page Website/`

A complete single-page website with smooth internal navigation linking to multiple sections.

**Key Features:**
- Sticky **navigation bar** with anchor links (`#home`, `#gallery`, `#video`, `#contact`)
- Responsive **CSS Grid** photo gallery with hover zoom effect
- Embedded **YouTube video** using an `<iframe>`
- **Contact form** with first name, email, and message fields
- Semantic structure using `<header>`, `<section>`, and `<footer>`

---

### 4. Basic HTML and CSS Exercises
**Folder:** `Basic_HTML_and_CSS/`

A collection of foundational exercises covering core HTML and CSS concepts.

| File | Description |
|---|---|
| `Basic_HTML_3.html` | Basic HTML document structure practice |
| `Basic_HTML_4.html` | HTML elements and attributes |
| `Basic_HTML_5.html` | Additional HTML structure exercises |
| `HTML_Table.html` | Creating and styling HTML tables |
| `HTML_Lists.html` | Ordered and unordered list structures |
| `form_assignment.html` | HTML forms with inputs, labels, and a submit button |
| `Basic_CSS_1.css` | Introductory CSS styling properties |
| `Basic_CSS_2.css` | CSS selectors and the box model |
| `CSS_Table.css` | Styling HTML tables with CSS |
| `CSS_Lists.css` | Styling ordered and unordered lists |
