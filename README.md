# Bootstrap 4 Project — Lumina Agency

A premium creative agency website built with **Bootstrap 4**, HTML5, CSS3, and vanilla JavaScript.

## 🌐 Live Preview

Open `index.html` in your browser to view the site.

## 📁 Project Structure

```
bootstrap4_project/
├── index.html          # Main webpage
├── css/
│   └── style.css       # Custom styles (glassmorphism, animations, dark theme)
├── js/
│   └── main.js         # Smooth scroll, scroll animations, form validation
└── README.md
```

## ✅ Bootstrap 4 Features Used

- **Navbar** — Responsive collapsible navbar (`navbar-expand-lg`, `navbar-toggler`, `collapse`)
- **Grid System** — `container`, `row`, `col-md-*`, `col-lg-*` throughout all sections
- **Cards** — Service cards, portfolio cards, team cards
- **Utility Classes** — `text-center`, `d-flex`, `justify-content-center`, `align-items-center`, `mb-*`, `mr-*`, `ml-auto`
- **Forms** — Contact form with labels and grouped inputs
- **Badges / Tags** — Section labels
- **Responsive Design** — Works on mobile, tablet, and desktop

## 📄 Pages / Sections

| Section    | Description                              |
|------------|------------------------------------------|
| Hero       | Full-height intro with stats and CTA     |
| Services   | 6 service cards in a Bootstrap grid      |
| Portfolio  | 6 project cards with hover overlays      |
| Team       | 4 team member cards with social links    |
| Contact    | Info + contact form with validation      |
| Footer     | Brand, links, newsletter signup, socials |

## 🛠 Technologies

- **Bootstrap 4.5.2** (CDN)
- **jQuery 3.5.1** (Bootstrap dependency)
- **Font Awesome 5** (icons)
- **Google Fonts** — Outfit + Inter
- Vanilla CSS3 (custom properties, animations, glassmorphism)
- Vanilla JavaScript (ES5 compatible)
